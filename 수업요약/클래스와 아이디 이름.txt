클래스(Class) 이름 예시와 설명:
.navigation: 웹 페이지의 주요 네비게이션 바를 나타냅니다. 사용자가 사이트를 탐색하는 데 사용됩니다.
.logo: 웹 페이지의 로고 이미지를 식별하는 클래스입니다. 회사 또는 브랜드를 나타냅니다.
.menu: 네비게이션 바에서 메뉴 항목들을 감싸고 정렬하는 컨테이너를 나타냅니다.
.dropdown: 드롭다운 메뉴에 스타일을 적용하는 클래스입니다. 추가 옵션을 제공하고 사용자가 선택할 수 있도록 합니다.
.search-bar: 검색 기능을 포함한 검색 바에 스타일을 적용하는 클래스입니다. 사용자가 원하는 콘텐츠를 검색할 수 있도록 합니다.
.slider: 이미지나 콘텐츠를 슬라이드 형태로 표시하는 요소에 사용됩니다.
.carousel: 이미지나 콘텐츠를 순환적으로 보여주는 카루셀에 사용됩니다.
.feature: 주요 기능이나 서비스를 강조하기 위한 요소에 사용됩니다.
.testimonial: 고객의 평가나 추천을 표시하는 요소에 사용됩니다.
.pricing-table: 제품이나 서비스의 가격을 비교하여 나타내는 표에 사용됩니다.
.team-member: 팀 구성원을 나타내는 요소에 사용됩니다. 각 팀원의 사진과 정보를 표시할 수 있습니다.
.gallery: 이미지나 비디오 등을 그룹화하여 보여주는 갤러리에 사용됩니다.
.widget: 사이드바나 푸터 등에 추가 정보나 기능을 제공하는 작은 모듈에 사용됩니다.
.sidebar: 웹 페이지의 사이드바를 식별하는 클래스입니다. 부가 정보나 관련 링크 등이 포함될 수 있습니다.
.tag: 콘텐츠에 대한 태그를 나타내는 요소에 사용됩니다. 주제를 분류하거나 검색을 용이하게 할 수 있습니다.
.badge: 특정 상태나 속성을 나타내는 작은 텍스트나 아이콘에 사용됩니다. 예를 들어, 'New', 'Hot', 'Sale' 등을 표시할 때 사용됩니다.
.alert: 사용자에게 중요한 메시지나 경고를 표시하는데 사용됩니다. 주로 빠른 주의를 요하는 상황에서 사용됩니다.
.modal: 모달 다이얼로그를 나타내는 요소에 사용됩니다. 사용자의 상호작용을 위해 팝업 창처럼 표시됩니다.
.popup: 웹 페이지에서 나타나는 작은 팝업 창에 사용됩니다. 주로 광고나 추가 정보를 표시할 때 사용됩니다.
.tooltip: 요소에 대한 추가 정보를 표시하기 위해 마우스 오버 시 나타나는 도움말 툴팁에 사용됩니다.
.popover: 요소 위에 마우스를 가져가면 컨텍스트 관련 정보가 팝업되는 팝오버에 사용됩니다.
.tab: 다중 탭 인터페이스를 나타내는 요소에 사용됩니다. 사용자가 탭을 선택하여 콘텐츠를 전환할 수 있습니다.
.accordion: 아코디언 스타일의 접히고 펼쳐지는 콘텐츠 영역을 나타내는 요소에 사용됩니다.
.progress-bar: 작업 진행 상태를 시각적으로 표시하는 데 사용됩니다. 주로 프로그레스 바나 스킬 바로 사용됩니다.
.timeline: 일련의 이벤트를 시간 순서대로 표시하는 요소에 사용됩니다. 역사, 이력, 이벤트 등을 표시하는 데 사용됩니다.
.card: 콘텐츠를 감싸고 시각적으로 그룹화하는 요소에 사용됩니다. 블로그 포스트, 제품, 카드 뉴스 등을 표시할 수 있습니다.
.avatar: 사용자나 프로필 이미지를 나타내는 요소에 사용됩니다.
.button: 버튼 요소에 스타일을 적용하는 클래스입니다. 사용자의 상호작용을 유도하거나 특정 작업을 실행합니다.
.icon: 아이콘 요소에 스타일을 적용하는 클래스입니다. 각종 아이콘을 나타내는 데 사용됩니다.
.social-media: 소셜 미디어 아이콘이나 링크를 포함하는 요소에 사용됩니다.
.footer: 웹 페이지의 하단 영역을 나타냅니다. 저작권 정보, 연락처 정보 등이 포함될 수 있습니다.
.breadcrumb: 사용자가 현재 페이지의 위치를 나타내는 경로를 표시하는데 사용됩니다.
.pagination: 긴 목록이나 페이지를 나타낼 때 페이지 네비게이션을 위해 사용됩니다.
.form: 폼 요소에 스타일을 적용하는 클래스입니다. 사용자의 입력을 받는데 사용됩니다.
.input: 입력 필드에 스타일을 적용하는 클래스입니다. 텍스트 박스, 체크박스, 라디오 버튼 등에 사용됩니다.
.textarea: 다중 행 텍스트 입력 필드에 스타일을 적용하는 클래스입니다.
.checkbox: 체크박스 요소에 스타일을 적용하는 클래스입니다.
.radio: 라디오 버튼 요소에 스타일을 적용하는 클래스입니다.
.select: 드롭다운 선택 메뉴에 스타일을 적용하는 클래스입니다.
.datepicker: 날짜 선택기를 나타내는 요소에 사용됩니다. 사용자가 달력에서 날짜를 선택할 수 있습니다.
.spinner: 입력 값의 증가 또는 감소를 표시하는 스피너 요소에 사용됩니다.
.badge: 특정 상태나 속성을 나타내는 작은 텍스트나 아이콘에 사용됩니다. 예를 들어, 'New', 'Hot', 'Sale' 등을 표시할 때 사용됩니다.
.alert: 사용자에게 중요한 메시지나 경고를 표시하는데 사용됩니다. 주로 빠른 주의를 요하는 상황에서 사용됩니다.
.tag: 콘텐츠에 대한 태그를 나타내는 요소에 사용됩니다. 주제를 분류하거나 검색을 용이하게 할 수 있습니다.
.chip: 작은 요소를 나타내며, 주로 사용자나 데이터 등을 나타냅니다.
.tooltip: 요소에 대한 추가 정보를 표시하기 위해 마우스 오버 시 나타나는 도움말 툴팁에 사용됩니다.
.popover: 요소 위에 마우스를 가져가면 컨텍스트 관련 정보가 팝업되는 팝오버에 사용됩니다.
.modal: 모달 다이얼로그를 나타내는 요소에 사용됩니다. 사용자의 상호작용을 위해 팝업 창처럼 표시됩니다.
.dialog: 대화 상자를 표시하는 요소에 사용됩니다. 모달과 비슷하지만 모달보다 더 많은 컨트롤을 제공합니다.
.popup: 웹 페이지에서 나타나는 작은 팝업 창에 사용됩니다. 주로 광고나 추가 정보를 표시할 때 사용됩니다.
.banner: 큰 배너 이미지나 광고를 표시하는데 사용됩니다. 주로 웹 페이지의 상단에 배치됩니다.
.promotion: 프로모션 콘텐츠를 나타내는 요소에 사용됩니다. 할인 정보나 이벤트를 강조하기 위해 사용될 수 있습니다.
.notification: 사용자에게 중요한 메시지를 전달하는데 사용됩니다. 새로운 알림이나 이벤트를 표시할 때 사용됩니다.
.featurette: 제품이나 서비스의 특징을 강조하는데 사용됩니다. 주요 기능을 나타내거나 소개하는 데 사용됩니다.
.testimonial: 고객의 평가나 추천을 표시하는 요소에 사용됩니다. 회사나 제품에 대한 긍정적인 의견을 나타냅니다.
.quote: 인용구를 표시하는데 사용됩니다. 주로 사람들의 명언이나 긍정적인 메시지를 표시할 때 사용됩니다.
.avatar: 사용자나 프로필 이미지를 나타내는 요소에 사용됩니다. 프로필 사진을 표시하는 데 사용됩니다.
.thumbnail: 작은 이미지를 표시하는데 사용됩니다. 이미지 갤러리나 카드에서 썸네일 이미지를 표시하는 데 사용됩니다.
.article: 블로그 포스트나 기사를 나타내는 요소에 사용됩니다. 글의 내용이나 메인 콘텐츠를 감싸고 있습니다.
.blog-post: 블로그 포스트를 나타내는 요소에 사용됩니다. 블로그의 글 내용이나 메타 정보를 표시하는 데 사용됩니다.
.comment: 사용자가 작성한 댓글을 나타내는 요소에 사용됩니다. 댓글 내용이나 작성자 정보를 표시하는 데 사용됩니다.
.reply: 댓글에 대한 답글을 나타내는 요소에 사용됩니다. 대댓글 형태로 표시될 수 있습니다.
.newsletter: 뉴스레터를 구독할 수 있는 폼을 나타내는 요소에 사용됩니다. 이메일 주소를 입력하여 구독할 수 있습니다.
.subscribe: 콘텐츠나 서비스를 구독할 수 있는 옵션을 제공하는 요소에 사용됩니다. 주로 뉴스레터나 업데이트 등을 구독할 때 사용됩니다.
.share: 콘텐츠를 소셜 미디어에 공유하는 옵션을 제공하는 요소에 사용됩니다.
.like: 사용자가 콘텐츠를 좋아하는 옵션을 제공하는 요소에 사용됩니다. 주로 소셜 미디어에서 사용됩니다.
.favorite: 사용자가 즐겨찾기에 추가할 수 있는 옵션을 제공하는 요소에 사용됩니다. 즐겨찾기 목록에 추가되는 버튼 등에 사용됩니다.
.rating: 제품이나 서비스에 대한 평가를 나타내는 요소에 사용됩니다. 별점이나 숫자 등으로 표시될 수 있습니다.
.review: 제품이나 서비스에 대한 리뷰를 나타내는 요소에 사용됩니다. 고객이 작성한 리뷰 내용을 표시할 수 있습니다.
.order: 주문을 나타내는 요소에 사용됩니다. 상품 주문 정보나 주문 상태를 표시하는 데 사용됩니다.
.cart: 쇼핑 카트를 나타내는 요소에 사용됩니다. 사용자가 장바구니에 담은 상품을 표시하는 데 사용됩니다.
.checkout: 결제 및 주문 확인 단계를 나타내는 요소에 사용됩니다. 주문 정보를 입력하고 결제하는 단계를 표시하는 데 사용됩니다.
.payment: 결제 방법을 선택하고 입력하는 폼을 나타내는 요소에 사용됩니다. 신용카드 정보나 결제 수단을 입력하는 데 사용됩니다.
.invoice: 송장을 나타내는 요소에 사용됩니다. 구매한 상품이나 서비스에 대한 비용과 세부 정보를 표시하는 데 사용됩니다.
.receipt: 영수증을 나타내는 요소에 사용됩니다. 주문한 상품이나 서비스에 대한 구매 내역을 표시하는 데 사용됩니다.
.profile: 사용자 프로필을 나타내는 요소에 사용됩니다. 사용자 정보나 설정을 표시하는 데 사용됩니다.
.settings: 사용자 설정을 관리하는 요소에 사용됩니다. 프로필 설정, 계정 설정 등을 표시하는 데 사용됩니다.
.account: 사용자 계정 정보를 표시하는 요소에 사용됩니다. 로그인 정보, 프로필 정보 등을 표시하는 데 사용됩니다.
.user: 사용자를 나타내는 요소에 사용됩니다. 사용자 프로필이나 로그인 정보 등을 포함할 수 있습니다.
.login: 로그인 폼을 나타내는 요소에 사용됩니다. 사용자가 로그인 정보를 입력하여 인증하는 데 사용됩니다.
.register: 회원가입 폼을 나타내는 요소에 사용됩니다. 새로운 사용자가 계정을 만들 때 사용됩니다.
.forgot-password: 비밀번호를 잊어버렸을 때 비밀번호를 재설정할 수 있는 요소에 사용됩니다.
.reset-password: 비밀번호를 재설정하는 폼을 나타내는 요소에 사용됩니다. 새로운 비밀번호를 입력하여 변경할 수 있습니다.
.verification: 사용자 이메일 주소나 전화번호를 인증하는 폼을 나타내는 요소에 사용됩니다.
.privacy-policy: 개인정보 보호 정책을 나타내는 페이지에 사용됩니다. 웹 사이트의 개인정보 수집 및 보호 정책을 설명합니다.
.terms-of-service: 이용약관을 나타내는 페이지에 사용됩니다. 웹 사이트의 이용약관을 설명합니다.
.copyright: 저작권 정보를 나타내는 요소에 사용됩니다. 웹 사이트의 콘텐츠에 대한 저작권 정보를 표시합니다.
.disclaimer: 책임 한계 및 면책 조항을 설명하는 페이지에 사용됩니다. 웹 사이트의 법적 고지 사항을 표시합니다.
.faq: 자주 묻는 질문(FAQ)을 표시하는 페이지에 사용됩니다. 웹 사이트의 자주 묻는 질문과 답변을 표시합니다.
.contact: 연락처 정보를 제공하는 페이지에 사용됩니다. 고객 서비스에 연락할 수 있는 정보를 제공합니다.
.about: 회사나 브랜드에 대한 정보를 제공하는 페이지에 사용됩니다. 회사의 소개나 미션 등을 설명합니다.
.mission: 회사나 브랜드의 미션을 설명하는 페이지에 사용됩니다. 회사의 목표나 비전을 제시합니다.
.vision: 회사나 브랜드의 비전을 설명하는 페이지에 사용됩니다. 장기적인 목표나 미래 전망을 제시합니다.
.values: 회사나 브랜드의 가치관을 설명하는 페이지에 사용됩니다. 핵심 가치나 신념을 나타냅니다.
.history: 회사나 브랜드의 역사를 설명하는 페이지에 사용됩니다. 회사의 과거와 성장 과정을 제시합니다.
.team: 회사나 조직의 팀 구성원을 소개하는 페이지에 사용됩니다. 팀 멤버들의 정보와 역할을 제시합니다.
