컬럼(Column) : 테이블을 구성하는 각각의 열(세로)에 위치한 정보
쿼리(Query)(질의어) =  SQL(Structured Query Language)
: 데이터베이스에 정보를 요청하는 것(DB에 대해 명령문)

서브쿼리는 메인쿼리 어느절에도 올 수 있지만 일반적으로 where절에 쓴다.

-- DESC : 내림차순(Descending)으로 정렬

CRUD는 이미 만들어진 데이터를 수정하는 용도로 쓰인다.

테이블을 수정한 후
-- ROLLBACK; :커밋 전에는 롤벡이 가능하지만,
 디비버는 오토커밋이 되기때문에 롤백이 불가능(데이터베이스창에 트랜잭션모드 누르면 변경 가능)
-- COMMIT; : 커밋하는 명령어(커밋이 안 된 상태에서 수정하는 경우 로딩이 걸리고 넘어가지 않음)
-- DELETE FROM 테이블이름; : 테이블 안에있는 데이터 삭제
-- DROP TABLE 테이블 이름; : 테이블 자체를 삭제
